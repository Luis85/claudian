The main idea of work-orders is to hand-off work and have it executed in the background. As of right now, the work-order tabs are to prominent in the tabs row in the chat sidebar, it would be better if the integrate in the same row as quick-actions and open a drop-down menu. The drop-down toggle should indicate when work-orders are running or need input, the drop-down menu itself shall indicate the current state of a dispatched/running work-order.
