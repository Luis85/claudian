To improve management capabilities, the user shall be able to click on a columns name and get it's content presented inside an Obsidian Base view.

The idea is to have column settings act as a bases view with all filters and properties set according to the agent-boards lane setup.

When board settings get changed/updated the underlying base must be synced.
